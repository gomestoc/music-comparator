Como criar a app Spotify
1
Vai a developer.spotify.com/dashboard e faz login com a tua conta Spotify.
2
Clica em "Create app". Preenche qualquer nome e descrição.
3
Em Redirect URI, introduz o URL desta página (o endereço que aparece no browser).
4
Em APIs used, seleciona Web API.
5
Guarda → vai a Settings → copia o Client ID e cola no separador Configurar.
Como criar a app Google Drive
1
Vai a console.cloud.google.com e cria um projeto novo.
2
Ativa a Google Drive API em "APIs e serviços".
3
Cria credenciais → OAuth 2.0 Client ID → tipo Aplicação Web.
4
Adiciona o URL desta página como Authorized redirect URI.
5
Copia o Client ID e cola no separador Configurar.
📲 Instalar como app Android
1. Abre esta página no Chrome no teu Android.
2. Toca nos 3 pontos no canto superior direito.
3. Seleciona "Adicionar ao ecrã principal".
4. Confirma — aparece o ícone como uma app normal! 🎉
